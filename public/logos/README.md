# Hiring-association logos

Drop the official company logo files here (PNG with transparent background, or
SVG) using the exact filenames below. They power the "Hiring associations"
strip on the home page.

Until a file is present, the chip automatically falls back to the Clearbit logo
CDN, and then to the company name as text — so the site always looks clean.

Recommended: wide/horizontal logo, transparent background, ~28px tall when
displayed (so export at 2x, e.g. ~56px tall / 200px+ wide).
